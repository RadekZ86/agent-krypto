Co to jest Agent Krypto?

Agent Krypto to osobisty asystent tradingowy, ktory automatycznie analizuje rynek kryptowalut i podejmuje decyzje kupna/sprzedazy.

  • Tryb PAPER — symulowany handel wirtualnym kapitalem. Bez ryzyka. Idealny do nauki i testowania strategii.
  • Tryb LIVE — agent wysyla prawdziwe zlecenia na Twoje konto Binance. Wymaga klucza API.

Agent analizuje wskazniki techniczne (RSI, MACD, trend, wolumen), uczy sie na wlasnych blędach i wykorzystuje AI do oceny rynku. Kazdy cykl (co 2 minuty) agent sprawdza rynek i decyduje: BUY, SELL lub HOLD.

Jak to dziala?

1
Pobranie danych rynkowych

Agent pobiera ceny, wolumen i dane OHLCV z Binance co 15 minut.

2
Analiza techniczna

Oblicza RSI, MACD, srednie kroczace (EMA), zmiane wolumenu, rozpoznaje formacje swieczek.

3
Silnik decyzyjny

Laczy wskazniki z regułami agenta (ostrozny / normalny / ryzykowny) i generuje decyzje BUY/SELL/HOLD z poziomem pewnosci.

4
Realizacja zlecen

W trybie PAPER — symulacja. W trybie LIVE — zlecenie market na Binance Spot.

5
Nauka i adaptacja

Agent analizuje wyniki swoich decyzji, uczy sie z bledow i dostosowuje strategie.

Jak podlaczyc konto Binance? (krok po kroku)

A. Zalozenie konta na Binance (jesli jeszcze nie masz)

1
Wejdz na binance.com/pl/register

Jesli jestes w Polsce, zostaniesz przekierowany na Binance Polska. Podaj email i hasło.

2
Weryfikacja tozsamosci (KYC)

Binance wymaga weryfikacji — przygotuj dowod tozsamosci (dowod osobisty lub paszport). Proces trwa 5–15 minut.

3
Wplac srodki

Kliknij "Wplac" → wybierz PLN → przelew BLIK lub bankowy. Minimum to ok. 10 PLN, ale zalecamy min. 100–200 PLN na start.

B. Utworzenie klucza API na Binance

1
Zaloguj sie na binance.com

Na komputerze (na telefonie tez mozliwe, ale latwiej na duzym ekranie).

2
Przejdz do Zarzadzania API

Kliknij ikone profilu (prawy gorny rog) → "Zarzadzanie API" (lub wejdz bezposrednio na binance.com/pl/my/settings/api-management).

3
Kliknij "Utworz klucz API"

Wybierz typ: "Klucz API wygenerowany przez system" (nie RSA / Ed25519). Nadaj nazwe np. AgentKrypto.

4
Potwierdz weryfikacje

Binance poprosi o kod z email + ewentualnie 2FA (Google Authenticator / SMS).

5
WAŻNE — Skopiuj Secret Key natychmiast!

Po utworzeniu zobaczysz API Key i Secret Key. Secret Key wyswietla sie tylko raz — skopiuj go i zachowaj w bezpiecznym miejscu.

C. Konfiguracja uprawnien i IP klucza

1
Kliknij "Edytuj ograniczenia"

Przy nowo utworzonym kluczu API kliknij "Edytuj ograniczenia".

2
Wlacz uprawnienia

Zaznacz: ✅ Wlacz odczyt (Enable Reading) + ✅ Wlacz handel Spot (Enable Spot Trading).

❌ NIE zaznaczaj: Wypłaty / Withdrawals / Futures / Margin — dla bezpieczenstwa.

3
Ogranicz dostep po IP

Wybierz "Ogranicz dostep tylko do zaufanych adresow IP" i wpisz:

77.79.248.3

To IP serwera Agent Krypto. Dzieki temu nikt inny nie bedzie mogl uzyc Twojego klucza nawet jesli go pozna.

4
Zapisz zmiany

Kliknij "Zapisz" i potwierdz kodem weryfikacyjnym.

D. Wklejenie kluczy w Agent Krypto

1
Zaloguj sie do Agent Krypto

Wejdz na agentkrypto.apka.org.pl i zaloguj się (lub zarejestruj nowe konto).

2
Otworz ustawienia API

Kliknij ikone uzytkownika 👤 (prawy gorny rog) → z menu wybierz "KASA I API".

3
Kliknij "+ Dodaj klucz"

Pojawi sie formularz. Wypelnij pola:

  • API Key — wklej caly klucz z Binance (np. aB3xY7k...)
  • Secret Key — wklej secret (ten ktory wyswietlil sie tylko raz)
  • Uprawnienia — wybierz "Trading"
4
Kliknij "Zapisz"

Agent automatycznie sprawdzi polaczenie z Binance. Jesli klucz jest poprawny i ma uprawnienia do handlu, zobaczysz zielony status.

5
Przelacz na tryb LIVE

W menu uzytkownika 👤 kliknij "LIVE". Dashboard pokaże Twoje realne saldo z Binance.

E. Ustawienie alokacji (ile agent moze handlowac)

1
Przejdz do zakladki "Ustawienia"

Kliknij ikone ⚙ w bocznym menu (lub "Ustawienia" na mobilce).

2
Ustaw alokacje LIVE

W karcie "Alokacja LIVE" wybierz ile agent moze uzywac przy kazdym zleceniu:

  • Procent portfela — np. 10% wolnych srodkow (domyslnie)
  • Dokladna kwota — np. 50 PLN na kazde zlecenie
  • Maksimum — agent uzyje caly dostepny balans (ostroznie!)

Tryby agenta

Ostrozny

Max 6 wejsc dziennie, 5 otwartych pozycji, 5% eksploracji. Najmniejsze ryzyko.

Normalny

Max 12 wejsc dziennie, 10 pozycji, 12% eksploracji. Zbalansowane podejscie.

Ryzykowny

Max 20 wejsc dziennie, 15 pozycji, 25% eksploracji. Wiecej probok = wiecej nauki, ale wieksze ryzyko.

Kryptowaluty a polskie prawo

Podatek od zyskow (PIT)

  • Zyski z kryptowalut podlegaja opodatkowaniu stawka 19% PIT (art. 30b ustawy o PIT).
  • Podatek rozliczasz raz w roku w zeznaniu PIT-38, termin: 30 kwietnia nastepnego roku.
  • Przychod powstaje w momencie zamiany krypto na walute fiat (np. sprzedaz BTC za PLN) lub zamiany krypto na krypto.
  • Kosztem uzyskania przychodu jest cena zakupu krypto (prowizje Binance tez mozna odliczyc).
  • Strate mozna rozliczyc w ciagu 5 nastepnych lat podatkowych.

Obowiazki ewidencyjne

  • Musisz prowadzic ewidencje transakcji — daty, kwoty, kursy, prowizje.
  • Agent Krypto zapisuje historie transakcji, ale zalecamy eksport i archiwizacje na wlasna reke.
  • Binance udostepnia historie transakcji do pobrania (CSV) — przechowuj ja min. 5 lat.

AML / KYC (przeciwdzialanie praniu pieniedzy)

  • Binance dziala w Polsce jako zarejestrowany VASP (dostawca uslug w zakresie aktywow wirtualnych) w rejestrze GIIF.
  • Weryfikacja tozsamosci (KYC) na Binance jest obowiazkowa — bez niej nie dokonasz transakcji.
  • Transakcje powyzej 15 000 EUR (rownowartosc) moga zostac zgloszone do GIIF.

Regulacje gieldowe

  • Kryptowaluty nie sa w Polsce uznawane za instrumenty finansowe ani za prawny srodek platniczy.
  • KNF nie nadzoruje rynku kryptowalut, ale ostrzega przed ryzykiem — inwestujesz na wlasna odpowiedzialnosc.
  • Od 2024 r. w UE obowiazuje rozporzadzenie MiCA (Markets in Crypto-Assets), ktore wprowadza licencjonowanie platform krypto.

Agent Krypto a prawo

  • Wszystkie decyzje tradingowe podejmujesz na wlasna odpowiedzialnosc.

Najczesciej zadawane pytania (FAQ)

Czy moge stracic pieniadze?

Tak. Handel kryptowalutami wiaze sie z wysokim ryzykiem. Agent nie gwarantuje zyskow. W trybie PAPER ryzykujesz tylko wirtualny kapital.

Jaki minimalny kapital potrzebuje na Binance?

Minimalna wartosc zlecenia na Binance Polska to 25 PLN. Zalecamy min. 100–200 PLN na start.

Czy agent moze wyplacic moje pieniadze?

Nie. Klucz API nie powinien miec uprawnien do wyplat. Agent moze tylko kupowac i sprzedawac na rynku Spot.

Ile kosztuje AI Advisor?

AI Advisor korzysta z OpenAI API. Koszt zalezy od modelu — aktualnie GPT-4.1 nano, ok. 0.001–0.01 USD za zapytanie. Zuzycie widoczne w zakladce Status.

Czy moge uzywac agenta na telefonie?

Tak! Agent Krypto to aplikacja webowa (PWA) — dziala w przegladarce na kazdym urzadzeniu. Na iPhonie mozesz dodac ja do ekranu glownego.

Jak wylaczyc automatyczny handel?

Kliknij "Zatrzymaj scheduler" w gornym pasku. Agent przestanie automatycznie analizowac rynek i handlowac.

Czy moje klucze API sa bezpieczne?

Klucze sa szyfrowane na serwerze. Zawsze ograniczaj klucz API do IP serwera (77.79.248.3) i nie wlaczaj uprawnien do wyplat.
